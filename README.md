# RentWise Splitter

Multi-page web app that solves real roommate money conflicts.

## Problem
People sharing apartments often argue over unclear expense splits.

## Solution
RentWise provides:
- bill recording with flexible split groups,
- automatic debt optimization,
- clear settlement actions,
- explicit household finance rules.

## Pages
- `/` Overview
- `/bills` Bill management workspace
- `/settlements` Auto-calculated transfer plan
- `/rules` Agreement and prevention rules

## API
- `GET/POST /api/bills`
- `GET /api/settlements`

## UI Blocks
Built with **Ant Design** components (`Menu`, `Card`, `Table`, `Form`, `Timeline`, `Statistic`).
